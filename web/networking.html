Networking

A simple broadband/wireless/ethernet networking diagram

Computer Network

Two or more computers connected to each other form a computer network. Based on its size, a network can be classified as LAN, WAN, MAN. The medium used to connect them could be twisted pair, coaxial, wireless, etc. A network protocol is the language used by the systems to talk to each other. Some of the common protocols used are TCP/IP, Ethernet, ATM, etc. Client/Server is a computer network in which the server is a powerful system with lots of resources and the client is a comparatively less powerful. The client sends requests to the server, which processes it and sends back a reply. A server can accept requests from many clients simultaneously.

Local Area Network ( LAN )

Local Area Network (LAN) refers to a group of computers connected together through a wired or wireless medium. They can be confined to a building or a small campus. The number of users on the network could range from two to as many as a thousand or more. The main technologies used in building local area networks are Ethernet for PCs and AppleTalk for Macintosh systems. A LAN can be characterised by the following properties: Topology: The manner in which the machines or nodes are arranged on the network. Some examples of LAN topology are Tree, Ring, Star, etc., based on the shape the network takes. Protocols: It is the set of rules used by the nodes to communicate with each other. Media: The medium that is used to physically connect the nodes, in case of wired networks.

Wide area Network ( WAN )

The term Wide Area Network refers to networks dispersed over a large area. It is usually used to refer to public or shared networks, but they are sometimes privately owned too. The Internet can be seen as a collection of WANs. Networks can be broadly classified as LANs, MANs and WANs. A MAN is in between in size between LANs and WANs.

Storage Area Network ( SAN )

Storage Area Networks are high speed networks that are mainly used to store and share data. A SAN is normally found in large enterprise networks. Machines used for SAN are optimised for fast retrieval and storage of data. They use a high speed storage subsystem, and the network is built so that data is available to all nodes and servers present on the network. SAN uses optical fibre connectivity for extremely fast data transfer rates, as well as disk mirroring and automatic backup methods for complete data redundancy and reliability.

Peer to Peer

A direct connection between two computers over a network is called a peer to peer, or P2P, connection. Unlike the client server model, with P2P, both computers have client and server capabilities. P2P software is most popular for sharing files over the Internet. Large numbers of users come together to exchange and share files by using networks such as KaZaA and Gnutella. Before sharing files, users need to download and install the network's software. Some P2P networks require a central server which contains a database of users and files.

Client/Server

Client/ Server is an architecture that's used to describe a computing model in which a client program makes a request to a server program to serve up Web pages, files, etc. If more than one client program makes a request, the server fulfils them using criteria such as request priority, type of request, server load, etc. Client/ Server architecture is followed in almost all network transactions. Web servers, FTP servers, application server are some of the examples that follow client/ server architecture. In the case of the Internet, any system connected to it is a client computer and the system serving the Web pages and files to the user is the server machine. As compared to client systems, server machines are built using powerful hardware due to the simultaneous requests that they need to process.

Servers

In general, a server is a computer application that provides services to other applications, either within the same computer or to others via a network. The computer, where a server program runs, can also be called a server. In the client server architecture, a server is program that waits for and executes requests by a client. Applications on a computer may function as a client for some programs and as a server for others. One of the different types of servers are:

Application Servers:
These are dedicated servers, for running certain software applications. Audio/Video Servers:
They enable streaming multimedia contents from Web sites.
Chat Servers:
They enable a large number of users to exchange information in an environment with real time discussion capabilities.
Fax Servers:
This is a server that computers can connect to in order to send faxes.
FTP Servers:
These are some of the oldest types of servers. They enable the transfer of files between computers using a protocol called File Transfer Protocol.
IRC Servers:
These are servers that allow clients to connect and chat and share files between themselves. Many IRC servers connect to form an IRC network.
List Servers:
They are used to manage mailing lists.
Mail Servers:
They store and deliver e mail across networks and the Internet.
News Servers:
These are USENET servers that are used to deliver news within public groups.
Proxy Servers:
These are servers that connect to the Internet, in order to share an internet connection across a network of clients. They monitor traffic, filter requests and improve performance and security.
